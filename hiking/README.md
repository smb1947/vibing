# Trail Picker (Hiking Skill)

A two-phase decision engine to pick the best hiking trails for Kritikal Adventures.

## Features

- **Phase 1**: Extracts candidates from AllTrails.
- **Phase 2**: Layers real-time weather, snow, road conditions, and user history.

## Usage

- Open the `skill.md` to see the agentic workflow.
- Run `ranker.py` to test the scoring logic.
